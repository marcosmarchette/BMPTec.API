# 🏦 API Banco Chu S.A.

API RESTful para gerenciamento de contas e transferências bancárias.

## 📋 Funcionalidades

### ✅ Implementadas
- **Cadastrar Contas** - Criação de contas bancárias
- **Realizar Transferências** - Transferências entre contas (somente em dias úteis)
- **Gerar Extrato** - Consulta de movimentações por período

### 🔄 Em Desenvolvimento
- Consultar conta por ID
- Listar contas por usuário
- Listar transferências por conta

## 🚀 Como Executar

### Pré-requisitos
- .NET 10 SDK
- SQL Server
- Visual Studio 2026 ou VS Code
